# TP_Final_TCP

Proyecto final de la materia Redes.  
Implementa un servidor y un cliente TCP utilizando contenedores Docker y GitHub Actions.  
Las imágenes se construyen y se suben al repositorio de Docker Hub de **itestyr** automáticamente.  

## Estructura del proyecto
- `.github/workflows`: contiene el flujo automatizado de construcción.
- `server/`: contiene el Dockerfile del servidor TCP.
- `client/`: contiene el Dockerfile del cliente TCP.
